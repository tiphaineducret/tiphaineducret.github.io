🎯 Objectif du projet

Concevoir et développer un portfolio numérique personnel présentant mes projets, compétences et parcours en BUT Science des Données. L'objectif était triple : réaliser un travail réflexif sur mes apprentissages, apprendre les bases du HTML et CSS, et produire un site web professionnel hébergé sur GitHub Pages.

🛠️ Compétences mises en œuvre

  • Création d'un site web complet en HTML et CSS
  • Hébergement sur GitHub Pages et gestion de versions
  • Conception d'une interface responsive et accessible
  • Rédaction de contenus clairs et structurés
  • Travail réflexif sur ses propres apprentissages

📋 Résultats & Livrables

Un portfolio en ligne accessible à l'adresse tiphaineducret.github.io, présentant les projets des deux semestres, une section parcours personnel, un bilan de l'année et le CV téléchargeable.

HTML CSS GitHub Pages JavaScript Design web

✨ Apprentissages clés

Ce projet m'a permis de découvrir le développement web de façon autonome et concrète. J'ai appris que créer un bon site ne se résume pas au code : la hiérarchie de l'information, la lisibilité et la cohérence visuelle sont tout aussi importantes. Le travail réflexif sur mes propres projets m'a aussi aidée à mieux comprendre ce que j'avais vraiment appris cette année.

❤️ Ce que j'ai aimé

J'ai adoré voir le site prendre forme progressivement et pouvoir le personnaliser à mon image. C'est gratifiant de construire quelque chose de concret qui me représente vraiment, et de partager mon parcours de façon professionnelle.

😟 Défis rencontrés

Le principal défi était d'apprendre HTML et CSS en autonomie tout en produisant un résultat professionnel. Gérer la compatibilité entre les pages, le responsive design et les petits bugs d'affichage a demandé beaucoup de patience et de tests.

💡 Contexte : Projet individuel évalué sur l'ergonomie et la forme (10 pts), la présentation des projets (10 pts), la section "À propos" (10 pts) et le bilan personnel (10 pts). Restitution le 15 juin 2026.
